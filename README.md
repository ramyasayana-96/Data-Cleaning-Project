# Airbnb Data Cleaning Project

## Project Overview

This project demonstrates how to clean a real-world Airbnb dataset using Python and Pandas.

## Dataset

- Dataset: Airbnb Open Data
- Source: Kaggle

## Objectives

- Handle missing values
- Remove duplicate records
- Fix incorrect data types
- Compare before and after cleaning
- Export cleaned dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Google Colab
- GitHub

## Project Structure

```
Airbnb-Data-Cleaning-Project/
│
├── Data_Cleaning_Airbnb.ipynb
├── Airbnb_Open_Data.csv
├── Airbnb_Open_Data_Cleaned.csv
├── README.md
└── requirements.txt
```

## How to Run

1. Open the notebook in Google Colab.
2. Upload `Airbnb_Open_Data.csv`.
3. Run all notebook cells.
4. Download the cleaned dataset.

## Output

- Airbnb_Open_Data_Cleaned.csv
- Airbnb_Open_Data_Cleaned.xlsx
